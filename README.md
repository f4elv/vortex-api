vortex-api

API construída com **NestJS**, utilizando **Prisma ORM** e **PostgreSQL**, organizada nos módulos **User** e **Admin**, com autenticação JWT e arquitetura modular.  
Este back-end faz parte do projeto **Vortex**, servindo como base para autenticação, administração e gerenciamento de dados.

---

Sobre o Projeto

A **Vortex API** foi desenvolvida com foco em boas práticas de arquitetura backend usando NestJS.  
Além de fornecer CRUDs essenciais, ela implementa:

- Autenticação JWT
- Controle de acesso (usuário comum e administrador)
- Integração com banco PostgreSQL via Prisma ORM
- Estrutura modular e escalável

Este repositório contém **somente a API**. O front-end está em outro repositório.

---

Tecnologias Utilizadas

- **Node.js**
- **NestJS**
- **Prisma ORM**
- **PostgreSQL**
- **JWT (Autenticação)**
- Helmet, CORS, Interceptors, Guards
